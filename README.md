# Creepy Chats

## About

Creepy Chats is a short visual novel about a student who chats with a chatbot to get help with completing an English assignment. Depending on the choices you make, you may either escape the chatbot scot free, or end up possessed by the consciousness of the bot.

This is a game made with Ren'Py!!
